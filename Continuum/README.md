# Continuum Simulations

**Purpose —** Numerical validation of continuum-like behavior on discrete graph motifs and extraction of spectral-scaling coefficients from random-walk diffusion. The primary motif studied here is the 4D torus \(\mathbb{Z}^4\).

---

## Abstract

This repository provides tools to (i) validate candidate graph motifs against continuum consistency checks, (ii) perform finite-size scaling (FSS) to extract the asymptotic spectral dimension and leading coefficient \(C_1\), and (iii) measure higher-order correction coefficients \(C_2\) and \(C_3\). The pipeline relies on Monte Carlo random walks on periodic lattices of varying linear size \(L\) and uses bootstrap resampling to estimate uncertainties.
